  
# 共识机制（ghost）

  

- 作废块也有出块奖励，uncle block。

- 叔块奖励奖励的7/8。

- 最多包含2个叔父区块。

- 包含了区块获得1/32。

# 修改协议

  

- 区块部分辈分，只要是之前的未在最长链的区块，都算叔父区块。

- 最多有7辈分的叔。

- 为了尽快合并区块链。

- 解决临时性分叉。

- 防止篡改。

- 只能合并合法区块，包含了非法块就成为坏块。

  
![](../pic/Pasted%20image%2020240730155608.png)

  
  

# 奖励

- block reward：不会自动变化。
- gas free：
- 叔父区块无法包含交易。
- 叔父区块只查header，交易容易与主链冲突。
- 叔块包含的交易会在后面的主链上进行执行。
- 只有分叉后的第一个区块才奖励。
